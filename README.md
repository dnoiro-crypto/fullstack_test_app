# fullstack_test_app

A minimal Flutter app scaffolded with Android and Web support.

- Material 3 is enabled in `lib/main.dart` (`ThemeData(useMaterial3: true)`).
- Firebase not added.

Quick start (from this folder):

```powershell
cd fullstack_test_app
flutter pub get
# Run on Android device/emulator
flutter run
# Run on Web (Chrome)
flutter run -d chrome
```

Notes:
- Ensure Flutter SDK is installed and on `PATH`.
- To add Firebase later, follow the official FlutterFire docs.
